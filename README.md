Frontend Setup

Clone the repository:

```
git clone https://github.com/yourusername/rwanda-ar-heritage.git
```

cd rwanda-ar-heritage/frontend

Install dependencies:

```
npm install
```

# or

```
yarn install
```

Start the development server:

```
npm run dev
```

or

```
yarn run dev
```
